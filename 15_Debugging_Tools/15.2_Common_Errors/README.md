# 15.2 Common Errors

Common C programming errors and how to identify and fix them.

## Error Categories

### Syntax Errors
- Missing semicolons
- Unmatched braces
- Incorrect function declarations
- Type mismatches

### Runtime Errors
- Segmentation faults
- Buffer overflows
- Null pointer dereference
- Division by zero
- Stack overflow

### Logic Errors
- Off-by-one errors
- Incorrect loop conditions
- Wrong algorithm implementation
- Incorrect assumptions

### Memory Errors
- Memory leaks
- Double free
- Use after free
- Uninitialized memory access
- Buffer overruns
